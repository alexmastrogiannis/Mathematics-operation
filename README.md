# Mathematics-operation
Ο πολλαπλασιασμός μπορεί να αναπαρασταθεί, περισσότερο συναρπαστικά, μέσω κινούμενων σχεδίων, με την αξιοποίηση του λογισμικού αισθητικής έκφρασης και ανάπτυξης της δημιουργικότητας Revelation Natural Art (RNA). Το RNA είναι ένα εύχρηστο και φιλικό, ανοιχτού, κονστρουκτιβιστικού τύπου λογισμικό, που προωθεί και διεπιστημονικές συνδέσεις. Ισχυρό πλεονέκτημά του είναι η δημιουργία ταινιών κινουμένων σχεδίων, γεγονός που μπορεί να αποτελέσει για τους μαθητές ισχυρό κίνητρο για συμμετοχή στην κατασκευή των ταινιών αλλά και για μάθηση. Ειδικότερα, μπορούν να σχεδιαστούν πολλά ενσταντανέ κινήσεων, στα οποία ορίζεται ο χρόνος της καθυστέρησή τους, και αυτόματα με ένα μόνο κουμπί και με αξιοποίηση του μετεικάσματος, η ταινία προβάλλεται στην οθόνη του υπολογιστή! Εδώ δημιουργήθηκαν 58 συνολικά ενσταντανέ για τη δημιουργία της ταινίας, που αναφέρεται στον πολλαπλασιασμό.
Η διαδικαστική, αλγοριθμική γνώση του πολλαπλασιασμού είναι πιθανόν να ενισχυθεί μέσα από εικονιστικές, πολύχρωμες και κινούμενες εφαρμογές, οι οποίες μπορεί να παροτρύνουν τους μαθητές, μέσω της δημιουργίας θετικών κινήτρων, ως απόρροιας μιας ενθουσιαστικής μαθησιακής ατμόσφαιρας.

Κονστρουκτιονισμός, αλγοριθμική Αριθμητική και λογισμικά αισθητικής έκφρασης και ανάπτυξης της δημιουργικότητας: Μια μαθησιακά αποδοτική συστράτευση. i-Teacher, ISSN 1792-4146, Μάιος 2015, τεύχος 11, σσ. 186-201".
